## 🚀 Quick Start

```bash
# Clone the magic
git clone https://github.com/NodeRoll/self-hosted-standalone.git

# Enter the portal
cd self-hosted-standalone

# Power up
npm install

# Configure your realm
cp .env.example .env

# Launch! 🚀
npm run dev
```
